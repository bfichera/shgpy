shgpy.shg_symbols module¶

This module defines a number of sympy.Symbol objects which are used throughout shgpy. They are used to define free parameters in suscepitbility tensors (see tensor_definitions), and to standardize name conventions for important variables (like phi and theta, which are used throughout shgpy as the standard azimuthal and incidence angles, respectively).

These variables in this module should be treated as protected and not modified by the user unless absolutely necessary.
