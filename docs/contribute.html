How to contribute¶

Bug reports¶

All of the major development of ShgPy happens on GitHub. To submit a bug report, just create an issue there.

Feature requests¶

Feature requests are welcome and encouraged! If there’s a particular feature you need or want (or if you’re interested in contributing code), please submit an issue on GitHub, or send me an email.
